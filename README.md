# MAL View
## MAL View Parser
A Scala ( sbt ) project.

Run `Main.scala` with IntelliJ ( required `Scala` plugin ).

Trace depth, trace domain and mock mode are set by `Config.scala`.

implementation hint → [README.md](./mal-view-parser/README.md)

## MAL View Viewer
A D3.js project.

Run server and show it with browser.

```
$ python -m SimpleHTTPServer 8080
```

http://localhost:8080

more info → [README.md](./mal-view-viewer/README.md)
